# Premium Cursor Effects ✨

A collection of modern cursor effects built using HTML, CSS, and JavaScript.

These effects are designed to make websites feel more interactive, premium, and visually engaging.

## Effects Included

* Neon Glow Ring Cursor
* Dot Chain Cursor
* Magnetic Snap Cursor
* Spotlight Reveal Cursor
* Gooey Elastic Blob Cursor

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Features

* Smooth animations
* Modern UI interactions
* Lightweight implementation
* Beginner-friendly structure
* Fully customizable

## Preview

Perfect for:

* Portfolio websites
* Landing pages
* Creative frontend projects
* Interactive UI experiences

## Getting Started

1. Clone the repository

```bash
git clone YOUR_REPOSITORY_LINK
```

2. Open the project folder

3. Run `index.html`
